FFMEPG

“ffmepg“是一个强大的多媒体处理的自由工具

配合ffmepg的视音merge与lux的视音download功能

Install

[Windows安装](https://github.com/iawia002/lux)(https://github.com/BtbN/FFmpeg-Builds)
[配置环境变量] -[./ffmepg.exe,lux.exe]

基本命令

lux %url% --默认下载
lux -p --专辑下载
lux BVxxxx1111 --以BV号下载
lux %1 --下载传入批处理形参链接
lux -o C:/download url --自定义位置
%windir%\system32\cmd.exe /c "lux.exe" 批处理调用

快捷方式

设置快捷方式的环境变量, 起始位置为%CD%指向工作目录
%windir%\system32\cmd.exe /c "lux.exe" -o --以快捷方式运行工具
