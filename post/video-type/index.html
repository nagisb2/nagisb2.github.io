计算机通用视频格式

发表于 2022-08-06- 更新于 2022-08-17
736- 2m-
文件

计算机通用“视频知识“,文件格式(MP4、MOV、WMV、FLV、AVI、WebM、MKV、DAT、M3U8)

码率/封装(容器)格式

  • 码率(1080p 60fps;30fps)

    • 文件体积与取样率成正比
    • 高码率越能减少视频失真

  • 封装

    • 视频格式也叫封装(容器)格式

    • AVC、Xvid、HEVC(视频编码); MP3、AAC(音频编码)

    • Xvid视频编码文件和MP3音频编码文件按AVI封装标准封装以后，得到AVI的封装格式视频(xxx.avi

MPEG4

  • MP4
    • 数字视频的标准,兼容性, 有损压缩,失真,适用于下载、上传和发送
    • 体积小,质量高,也适用于保存电影、蓝光文件等 ,支持主流浏览器

MOV

  • 影片格式
    • 无损压缩, 体积大于有损压缩, 苹果公司文件类型
    • 无法在网页上直接播放,支持游戏, 脚本封装到 MOV

WMV

  • Windows 媒体文件

    • 曾被利用Windows Media Playe来插入病毒

    • 一般同时包含视频和音频部分, 支持边下载边播放,Chrome安装插件来播放

FLV

  • 流媒体格式

    • 在线视频流媒体网站的理想选择(如:BiliBili, Youtube,土豆)
    • 文件极小、加载快, Flash动画的文件格式, 有损失真大,本地播放不理想,可塑性小

AVI

  • 音频视频交错格式

    • Windows3.1推出,压缩小,封装无损文件非常大(比MOV)
    • 兼容性较差(Microsoft设备，游戏机, Windows Media Player),多用于保存电影、多媒体光盘

WebM

  • 免版税的视听媒体格式

    • Google于2010年引入,基于VP8/9的开源编/解码器
    • 有损压缩(质量比MPEG略小),支持主流浏览器,Youtube ,HTML5视频元素

MKV

  • .mkv .mk3d .mka .mks

    • Matroska 适用于存储电影、电视剧、游戏特典,质量好,文件大(比MPEG-4)
    • 几乎支持任何视频和音频格式, 高度自适应性,易用性(比MPEG-4)
    • 本地播放效果好,Windows Media player需安装Matroska 解码分离器插件

DAT

  • 数据流格式

    • VCD光盘/ISO光盘的格式文件, 出现在游戏数据data/目录下
    • mpeg1编码，清晰度一般,Windows Media Player 11或将文件格式.dat改为mpeg来播放

M3U8

  • 索引文件

    • m3u8格式的文件存储多媒体播放列表,类似于快捷方式
    • 文件非常小,本质是文本文件,打开后将视频url下载后按顺序合并可以得到视频文件

MPEG2

  • .mpeg .mpg

    • 压缩率低, 质量较低(比mp4)
    • 兼容性很好,用于DVD制作, HDTV,视频录制,游戏OP/ED,可塑性好 -(比MPEG4)

后缀名

  • 文件扩展名
    • （Filename Extension)延伸文件名、后缀名）标志文件格式的一种机制
    • 视频改变后缀名只是改变格式的标识, 播放时系统会自动匹配解码分离器
